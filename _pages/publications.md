## Selected Publications

- **QAL: A Loss for Recall–Precision Balance in 3D Reconstruction** (WACV 2026) — Quality-aware loss that improves coverage and thin-structure recovery for point cloud reconstruction. [Project](https://droneslab.github.io/qal/)
- **Empir3D: A Framework for Multi-Dimensional Point Cloud Assessment** — Multi-metric evaluation of point clouds (resolution, accuracy, coverage, artifacts) beyond Chamfer/EMD. [arXiv](https://arxiv.org/abs/2306.03660)
- **A Modular, Extensible Framework for Modern Visual SLAM Systems** (MobiSys 2022 Poster) — Poster on a modular SLAM framework supporting extensible mapping and safe concurrency. [ACM DL](https://dl.acm.org/doi/abs/10.1145/3498361.3538793)
